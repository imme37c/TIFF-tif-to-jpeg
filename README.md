# TIFF-tif-to-jpeg
This tool wil convert your TIFF/tif files to jpeg!

# How to use mac? 
1. download tiff2jpeg.py and place it on your desktop (or anywhere you like)
2. install homebrew (terminal): /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
3. install python 3 (terminal): brew install python
4. install pillow (terminal): pip3 install pillow
5. navigate to desktop (or to the location where tiff2jpeg.py is located) (terminal): cd ~/desktop
6. run the python file! (terminal): python3 tiff2jpeg.py

# How to use windows?
1. download tiff2jpeg.py and place it on your desktop (or anywhere you like)
2. install Python 3 (Command Prompt / PowerShell): winget install Python.Python.3
3. install Pillow (Command Prompt / PowerShell): pip install pillow
4. navigate to desktop (or to the location where tiff2jpeg.py is located) (Command Prompt / PowerShell): cd %USERPROFILE%\Desktop
5. run the python file! (Command Prompt / PowerShell): python tiff2jpeg.py
