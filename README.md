# TIFF-tif-to-jpeg
This tool wil convert your TIFF/tif files to jpeg!

# How to use mac? 
-download tiff2jpeg.py and place it on your desktop (or anywhere you like)
-install homebrew (terminal): /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
-install python 3 (terminal): brew install python
-install pillow (terminal): pip3 install pillow
-navigate to desktop (or to the location where tiff2jpeg.py is located) (terminal): cd ~/desktop
-run the python file! (terminal): python3 tiff2jpeg.py

# How to use windows?
-windows
-download tiff2jpeg.py and place it on your desktop (or anywhere you like)
-install Python 3 (Command Prompt / PowerShell): winget install Python.Python.3
-install Pillow (Command Prompt / PowerShell): pip install pillow
-navigate to desktop (or to the location where tiff2jpeg.py is located) (Command Prompt / PowerShell): cd %USERPROFILE%\Desktop
-run the python file! (Command Prompt / PowerShell): python tiff2jpeg.py
